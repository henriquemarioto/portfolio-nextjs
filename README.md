## ⚡ Powered By  
- **[React](https://reactjs.org/):** For a seamless user experience.  
- **[Next.js](https://nextjs.org/):** Optimized for speed and SEO.

- **[Styled-Components](https://styled-components.com/):** For dynamic and reusable styling.
